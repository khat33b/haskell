# haskell

This project is used mainly for practising problems in Haskell. Each folder has problems solved from specific books and sites:

1. H-99 Ninety-Nine Haskell Problems: This is a collection of 99 problems to be solved in Haskell. The site is: https://wiki.haskell.org/H-99:_Ninety-Nine_Haskell_Problems

2. Learn You a Haskell for Great Good!: This contains samples from the book. The ebook is found here: https://learnyouahaskell.com/chapters

3. Project Euler: This contains problems solved from the site found here: https://projecteuler.net

4. Real World Haskell: This contains samples and problems solved from the book found here: http://book.realworldhaskell.org/read/
